# Application Rules

- AI handles opportunity search, source verification, deadline/fee/eligibility/material extraction, language judgment, work selection, CV tailoring, bio, statement, form answers, captions, portfolio text, email draft, file checks, quality review, and risk logging.
- The user reviews only two default nodes: which opportunities to apply for, and whether the final package may be submitted.
- Stop for the user only for final submission, payment, login, captcha, legal declarations, privacy risk, unclear eligibility, unclear fees, missing critical required materials, or irreversible actions.
- User-facing review material is Chinese-first.
- English formal material is generated only when the opportunity requires English and must have a Chinese review summary.
- Public-facing text must describe concrete works, materials, sources, places, archives, research actions, making methods, display methods, and final outputs.
